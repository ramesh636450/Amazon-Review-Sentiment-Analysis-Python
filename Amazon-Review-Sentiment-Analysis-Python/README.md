# Amazon Customer Review Sentiment Analysis – Python

## Project Overview
Performed sentiment analysis on Amazon customer reviews to identify customer opinions, product issues, and feedback trends using NLP techniques.

## Tools Used
- Python
- Pandas
- NLTK
- Matplotlib
- Seaborn
- WordCloud
- Jupyter Notebook

## Key Analysis Performed
- Review text preprocessing
- Sentiment classification
- Word frequency analysis
- Negative review analysis
- WordCloud generation

## Visualizations
- Sentiment distribution charts
- WordCloud visualization
- Top negative feedback keywords
- EDA charts

## Business Insights
- Identified common customer complaints and product issues.
- Analyzed positive and negative sentiment trends.
- Generated actionable recommendations to improve customer satisfaction.
